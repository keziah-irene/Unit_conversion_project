**Universal Unit Conversion** is a GUI-based Universal Unit Converter built using **Python and Tkinter** that allows users to convert values across multiple unit categories 
such as length, mass, temperature, speed, time, and data. This project separates **logic and UI**, making the code modular, readable, and easy to extend.

**Features:**
1. Converts units across 6 categories:
  * Length
  * Mass
  * Temperature
  * Speed
  * Time
  * Data
2. User-friendly GUI using Tkinter
3. Input validation for:
  * Positive numbers
  * Decimals
  * Negative values (for temperature)
4. Dynamic unit selection based on category

**Technologies Used**
1. Python 3
2. Tkinter (GUI framework)
3. ttk widgets

**How the Project Works:**
1. User selects a **category** (e.g., Length)
2. Corresponding **units are populated automatically**
3. User enters the **value**
4. Clicks **Convert**
5. The result is displayed instantly

**How to Run the Project**
1. Clone the repository
2. Navigate to the project folder
3. Run the application: python main.py

**Supported Conversions:**
1. Length: Kilometer, Meter, Centimeter, Millimeter, Foot, Inch
2. Mass: Kilograms, Grams, Milligrams, Pounds, Ounces
3. Temperature: Celsius, Fahrenheit, Kelvin
4. Speed: Meter/second, Kilometer/hour, Kilometer/second, Miles/hour
5. Time: Year, Month, Day, Hour, Minute, Second, Millisecond
6. Data: Terabyte, Gigabyte, Megabyte, Kilobyte, Byte
